# 🔐 CAMBIAR CONTRASEÑA DE ADMIN

## ¿Por Qué?

Por seguridad, la contraseña NO debe estar visible en documentos ni en el código.

---

## ✅ FORMA SEGURA

### Paso 1: Ejecutar el Script

En terminal, en la carpeta del proyecto:

```bash
python cambiar_contraseña.py
```

### Paso 2: Ingresar Contraseña Actual

```
Contraseña actual (deja en blanco si es la primera vez): 
```

- Si es la **primera vez**, solo presiona **Enter**
- Si ya tienes contraseña, escribe la actual

**La contraseña NO se verá mientras escribes** (es normal)

### Paso 3: Nueva Contraseña

```
Nueva contraseña: 
```

Escribe tu nueva contraseña segura (mínimo 4 caracteres)

**No se verá mientras escribes** ✓

### Paso 4: Confirmar

```
Confirmar nueva contraseña: 
```

Escribe de nuevo la misma contraseña

### Paso 5: ¡Listo!

```
✅ ¡Contraseña actualizada correctamente!
   Nueva contraseña: *****
```

---

## 💡 RECOMENDACIONES

✅ **Buenas contraseñas:**
- Mínimo 4 caracteres (mejor 8+)
- Mezcla números y letras
- Fácil de recordar para tu mamá
- Ejemplos: `momoprecio2024`, `supermarket123`, `tienda2024mt`

❌ **Evita:**
- Contraseñas muy cortas
- Tu fecha de nacimiento (fácil de adivinar)
- Números comunes (1234, 0000)

---

## 🆘 SI OLVIDAS LA CONTRASEÑA

Si olvidas la contraseña y no tienes acceso, ejecuta:

```bash
python cambiar_contraseña.py
```

- Presiona **Enter** en "Contraseña actual"
- Establece una nueva contraseña
- ¡Listo!

---

## 📱 PARA LA TABLET

**En el local:**
1. Conecta un teclado USB a la tablet
2. Ejecuta el script `python cambiar_contraseña.py`
3. Sigue los pasos

O desde tu portátil:
1. Ejecuta `python cambiar_contraseña.py`
2. Cambia la contraseña
3. Accede desde la tablet con la nueva

---

## ⚠️ IMPORTANTE

- **La contraseña NO se muestra** en pantalla (es seguridad)
- **NO la compartas** con nadie que no sea de confianza
- **Guarda en papel** en lugar seguro como respaldo
